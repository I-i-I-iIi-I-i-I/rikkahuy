## ✨ 本分支功能

- <img src="app/src/main/assets/icons/fal-color.svg" width="16" height="16"> **fal.ai 支持**
- <img src="app/src/main/assets/icons/claude-color.svg" width="16" height="16"> **Anthropic Claude 提示詞快取支援**
- 🛡️ **訊息混淆** — 西里爾文↔拉丁文，不可見字元，同形異義詞
- 🔇 **隱藏 TTS 按鈕選項**
- 👆 **滑動刪除聊天預覽中的訊息**
- 🐛 **錯誤修復**
  - 助手設置中滑塊無法移動
  - 不支持斜體的字體上斜體文本無法渲染
  - 助手設置中的輸入框存在延遲
  - 部分圖片裁剪不正確
